Think-Board is a full-stack note and task management application built to help developers organize learning goals, ideas, and daily practice in a clean and focused interface.

---

## Features

- Create, edit, and delete notes
- Clean dark-mode UI
- Full CRUD functionality
- Secure backend configuration
- RESTful API architecture
- Scalable project structure

---

## Tech Stack

**Frontend**
- React
- Tailwind CSS
- JavaScript (ES6+)

**Backend**
- Node.js
- Express.js
- MongoDB
- Mongoose
